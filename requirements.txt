# LectureForge — Python dependencies (RunPod / any CUDA box)

# NOTE on PyTorch: use a CUDA PyTorch base image (e.g. a RunPod "PyTorch" template)
# so torch/torchvision are already installed with the right CUDA build. This file
# deliberately does NOT pin torch — installing it here could pull a CPU-only build
# and clobber the pod's GPU torch. If your box has NO torch, install it first from
# https://pytorch.org matching your CUDA version, then `pip install -r requirements.txt`.
#
# Install:  pip install -r requirements.txt
# Also needs system packages (not pip):  apt-get install -y ffmpeg curl zstd
